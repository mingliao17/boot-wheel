# boot-wheel
脚手架springboot集成各类服务


- boot_axis2
> 集成webservice服务，努力中...
- boot_elasticsearch
> 集成elasticsearch全文检索服务，努力中...
- boot_exception
> 全局异常处理，目前只做了简单的全局配置和一个基础的异常体验全局异常处理
- boot_rabbitmq
> 集成ampq协议消息队列，目前只做了配置服务和一个基础的测试类
- boot_redis
> 集成缓存服务，目前只做了配置服务和一个基础的测试类
- boot_thymeleaf
> 集成泰姆利弗模板引擎，目前只加了一个layuimini的前端写好的架子
